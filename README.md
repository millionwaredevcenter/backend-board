# backend-board

### Description

```
이 예제는 밀리언웨어 개발센터 기본 교육을 위한 코드입니다.
spring boot 을 이용한 기본적인 게시판을 간단히 구현하는
교육용 코드입니다.
아래 라이브러리가 사용되었습니다.
JPA, H2, Devtools, Lombok
Enviroment :  JAVA , Maven
```
